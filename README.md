"more about the app"
# App that helps students or learners to keep track of their work.
#  It allows them to add their courses(as subjects) to the app.
#  Add sessions and give weekly summaries on how they worked for a particular week.
#  It gives the results in terms of percentage.
#  It is designed to make students/learners lives easy.

"Inspirations"
# The inspiration came from the law of consistency.
# Remember with good determination you can start ,but without consistency you wont finish right.
# The app helps you to track how good you are doing in terms of staying consistent with your work,weekly.
# Helps you decide reasonable hours to study a module/subject.
# Gives realistic results.

"How to use the App"
# The app uses the .json file to store the data that a student/learner has added.
# Everytime the app starts it loads data that a student/learner has.
# This data is stored in a .json file in ""DATA"" folder in the app folder.
# On a first run, the .json file wont be there but as soon as one adds data it gets saved.
# The app saves data automatically just after adding it.

"""What it has."""
# 1. Add module/subject, you need to decided how many hours you are going to study that particular
# module/subject , and scale its difficulty to 10.
# 2. Modify the subject/module you added , for instance you want to decrease or increase the 
# number of hours you want to study or its difficulty.
# 3. Add sessions , when studying 
#                - Enter the name of the module/subject you want to study.
#                - Total number of hours for that session.
#                - Focus level.
# 4. Get summaries.
#             1. - A current week summary.
#                - A particular module/subject
#             2. They are also automatical.
#                - After every 6 days starting from when you added the app, it gives summaries.
